# Операционная аналитика

Локальное браузерное приложение для сравнения операционных показателей между Excel-периодами.

Инструмент помогает быстро загрузить несколько выгрузок, сопоставить строки по выбранной колонке с объектом сравнения, выбрать нужные показатели, рассчитать динамику и выгрузить итоговый отчет.

Проект рассчитан на внутреннюю операционную аналитику: KPI, качество, ошибки, скорость обработки, AHT, CSAT, продажи и другие регулярные метрики.

## Главное

- Работает полностью локально в браузере.
- Не требует backend, базы данных, облака или API.
- Не отправляет Excel-файлы и данные в интернет.
- Не привязан к конкретным названиям столбцов.
- Пользователь сам выбирает колонку с объектом сравнения и показатели для анализа.
- Поддерживает сравнение двух и более периодов.
- Поддерживает CSV/Excel-экспорт, включая лист с графиками в Excel-отчете.

## Как запустить

Откройте файл `index.html` в браузере.

Установка зависимостей не нужна: используемые библиотеки уже лежат локально в папке `vendor/`.

Поддерживаемые входные форматы:

- `.xlsx`
- `.xls`
- `.csv`
- `.tsv`

## Основной сценарий

1. Загрузить файлы периодов.
2. При необходимости добавить дополнительные периоды через кнопку `+ Период`.
3. Выбрать формат сравнения.
4. Проверить preview таблиц.
5. Выбрать колонку с объектом сравнения для каждого периода.
6. Добавить один или несколько показателей.
7. Нажать `Сравнить`.
8. Изучить дашборд, график, лучшие и худшие изменения.
9. Скачать результат в CSV или Excel.

## Форматы сравнения

### Итоговый

Сравнивает последний период с первым:

```text
Период N - Период 1
```

Подходит, если нужно понять общий итог за несколько периодов.

### Последовательный

Сравнивает каждый следующий период с предыдущим:

```text
Период 2 - Период 1
Период 3 - Период 2
Период 4 - Период 3
```

Подходит для анализа динамики по неделям, месяцам или любым регулярным срезам.

## Что считается объектом сравнения

Объект сравнения — это ключ для сопоставления строк между файлами.

Им может быть:

- логин сотрудника;
- ФИО;
- табельный номер;
- команда;
- офис;
- направление;
- любой другой стабильный идентификатор строки.

Важно, чтобы выбранная колонка присутствовала во всех загруженных периодах.

## Возможности анализа

- расчет абсолютного изменения;
- расчет процентного изменения;
- автоматическое распознавание процентных показателей;
- отображение процентной динамики в процентных пунктах;
- поиск роста, падения и отсутствия изменений;
- вывод лучших и худших изменений;
- проверка отсутствующих объектов сравнения;
- проверка дублей;
- предупреждения о различающейся структуре файлов;
- подсветка проблем качества данных.

## Дашборд

Дашборд показывает:

- количество периодов;
- количество объектов сравнения;
- количество сопоставленных строк;
- количество найденных проблем;
- горизонтальный график динамики выбранного показателя;
- лучшие и худшие изменения по показателям;
- таблицу результатов.

График построен на Chart.js. Tooltip графика следует за курсором внутри активной полосы и показывает объект сравнения и динамику.

## Экспорт

### CSV

Экспортирует плоскую таблицу результатов.

### Excel

Формирует `.xlsx`-отчет с несколькими листами:

- `Обзор` — краткая сводка по анализу;
- `Сводка` — агрегированные показатели;
- `Сравнение` — детальная таблица сравнения;
- `Графики` — PNG-графики динамики по всем выбранным показателям;
- `Отсутствующие` — объекты сравнения, которые есть не во всех периодах;
- `Дубли` — повторяющиеся объекты сравнения и номера строк, где они найдены.

Excel-отчет формируется локально через ExcelJS.

## Назначение модулей

- `main.js` — связывает интерфейс, состояние приложения и бизнес-логику. Обрабатывает загрузку файлов, выбор колонок, запуск анализа и экспорт.
- `src/appState.js` — создает начальное состояние: периоды, выбранные показатели, режим сравнения, результаты и сообщения.
- `src/normalizers.js` — нормализует значения: пробелы, регистр, числа, проценты, пустые значения и форматирование вывода.
- `src/excelReader.js` — читает Excel/CSV/TSV через SheetJS, определяет строку заголовков, собирает preview и первичные предупреждения по структуре.
- `src/comparator.js` — сопоставляет строки по объекту сравнения, строит пары периодов, считает динамику, процент изменения, пропуски, дубли и нечисловые значения.
- `src/analytics.js` — готовит агрегаты для дашборда: количество объектов сравнения, проблем, улучшений, падений, лучшие и худшие изменения.
- `src/exporters.js` — формирует CSV и Excel-отчеты. Для Excel дополнительно создает изображение графика на canvas и вставляет его в отчет.
- `src/charts.js` — рисует график динамики через Chart.js и настраивает tooltip, цвета, анимацию и форматирование значений.
- `src/ui.js` — рендерит preview, выбор колонок, проверки качества данных, KPI-карточки, таблицу результатов и блоки дашборда.
- `src/motion.js` — добавляет аккуратные анимации появления секций, карточек и графиков через IntersectionObserver.

## Текущие ограничения

- Направление метрик пока не настраивается: положительная динамика считается ростом, отрицательная — падением.
- Таблица результатов ограничивает отображение первыми 300 строками, чтобы интерфейс оставался отзывчивым.
- Нет сохранения шаблонов сопоставления колонок.
- Нет фильтров и поиска по итоговой таблице.
- Нет отдельного режима для больших файлов с виртуализацией таблиц.
- Нет backend и базы данных по архитектурному решению MVP.
