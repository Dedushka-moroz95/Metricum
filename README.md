# Operational Analytics Dashboard

Локальный аналитический dashboard для сравнения показателей на основе Excel-файлов.

Проект создан как internal-tool для автоматизации рутинной аналитики операционного руководителя:
- сравнение показателей между периодами;
- анализ динамики;
- визуализация изменений;
- экспорт результатов.

--------------------------------------------------

ОСНОВНЫЕ ВОЗМОЖНОСТИ

[+] Сравнение Excel-файлов
- загрузка нескольких Excel-файлов;
- сравнение периодов (неделя к неделе, день к дню и тд);
- последовательный режим анализа.

[+] Сопоставление сотрудников
- выбор ID-колонки вручную;
- поддержка логинов, ФИО, команд;
- поиск пропущенных сотрудников;
- обработка дубликатов.

[+] Аналитика и динамика
- рост/падение показателей;
- процент изменения;
- абсолютное изменение;
- лучшие и худшие результаты;
- direction-aware comparison:
  некоторые метрики должны расти,
  некоторые — снижаться.

[+] Dashboard и графики
- визуализация KPI;
- графики динамики;
- summary cards;
- сравнительные таблицы.

[+] Экспорт
- экспорт в CSV;
- экспорт в Excel;
- формирование аналитических отчетов.

[+] Локальная обработка данных
Все вычисления происходят локально в браузере:
- без backend;
- без серверов;
- без отправки данных в интернет.

Это особенно важно для работы с внутренними корпоративными данными.

ОСОБЕННОСТИ ПРОЕКТА

[+] Flexible mapping
Проект не привязан к конкретным названиям колонок.

Пользователь самостоятельно выбирает:
- идентификатор сотрудника;
- нужные KPI;
- показатели для анализа.

[+] Нормализация данных
Система автоматически:
- очищает пробелы;
- преобразует проценты;
- нормализует числа;
- приводит значения к единому формату.

[+] Проверка качества данных
- поиск дубликатов;
- поиск отсутствующих сотрудников;
- обработка пустых значений;
- предупреждения о проблемах структуры.
